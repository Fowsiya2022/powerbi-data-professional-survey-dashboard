# Data Professional Survey Dashboard (Power BI)

## Overview

This project is an interactive Power BI dashboard built using survey data from data professionals. The goal of the project was to explore trends in salaries, job roles, programming languages, and job satisfaction within the data industry.

The dashboard provides insights into how professionals in different data roles perceive their careers, including salary expectations, work-life balance, and the difficulty of entering the data field.

## Tools Used

* Power BI
* Data Cleaning
* Data Visualization
* Data Analysis

## Dataset

The dataset comes from a survey conducted among data professionals and includes information such as:

* Job title
* Country
* Average salary
* Favorite programming language
* Work/life balance satisfaction
* Salary satisfaction
* Difficulty breaking into the data field

## Dashboard Features

The Power BI dashboard includes the following visualizations:

* **Country Distribution of Survey Takers** – Shows where respondents are located.
* **Average Salary by Job Title** – Compares salaries across data roles.
* **Favorite Programming Language** – Displays the most popular languages among data professionals.
* **Difficulty Breaking Into Data** – Shows how respondents perceive entry into the data industry.
* **Work/Life Balance Happiness Score**
* **Salary Happiness Score**

## Key Insights

* Data Scientists reported the highest average salaries among respondents.
* Python was the most popular programming language in the survey.
* Many respondents reported moderate difficulty breaking into the data field.
* Work-life balance satisfaction was higher than salary satisfaction for most participants.
